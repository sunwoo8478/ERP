# Security

## 민감 정보

- DB 비밀번호, JWT secret, SMTP 계정은 환경 변수로 관리합니다.
- `.env` 파일은 커밋하지 않습니다.
- 스크린샷에 실제 직원 정보나 급여 정보가 들어가지 않도록 확인합니다.

## 운영 전 필수 변경

- 개발 기본 관리자 계정 교체
- `JWT_SECRET`을 충분히 긴 난수로 교체
- PostgreSQL 계정 권한 최소화
- 운영 환경에서 `show-sql` 비활성화 검토
- 메일 계정은 운영 전용 계정으로 분리

## 공개 이슈 주의

취약점이나 인증 정보가 포함된 내용은 공개 이슈에 남기지 마세요. 공개 저장소에는 예시 데이터와 비식별 정보만 사용합니다.
