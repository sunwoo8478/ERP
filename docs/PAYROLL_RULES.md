# Payroll Rules

이 문서는 PayFit ERP의 급여 계산 관점을 정리합니다. 실제 운영에서는 회사 규정, 세법, 최신 보험 요율을 별도로 검증해야 합니다.

## 계산 순서

```text
1. 직원의 호봉 기준급여 조회
2. 개인별 급여 기준 수당 조회
3. 입사일 기준 월할 계산
4. 비과세 한도 적용
5. 과세소득 계산
6. 4대보험과 세금 공제
7. 연장·야간·휴일근로수당 반영
8. 무급휴가 공제
9. 실지급액 계산
10. 명세서와 지급·공제 항목 저장
```

## 지급 항목

| 항목 | 설명 |
| --- | --- |
| 기본급 | 직급, 호봉, 적용 연도 기준 급여 |
| 식대 | 개인별 급여 기준에 등록된 식대 |
| 교통비 | 자가 차량 여부와 비과세 한도에 따라 처리 |
| 직책수당 | 개인별 급여 기준에 등록된 직책 수당 |
| 연장근로수당 | 월 소정근로시간 기준 시급과 배율 적용 |
| 야간근로수당 | 승인된 야간근무 기록 기준 |
| 휴일근로수당 | 승인된 휴일근무 기록 기준 |

## 공제 항목

| 항목 | 계산 기준 |
| --- | --- |
| 국민연금 | 과세소득과 연금 기준 상한 적용 |
| 건강보험 | 과세소득과 건강보험 요율 |
| 장기요양보험 | 건강보험료와 장기요양 요율 |
| 고용보험 | 과세소득과 고용보험 요율 |
| 갑근세 | 간이세액표 성격의 계산 함수 |
| 지방소득세 | 갑근세의 10% |
| 무급휴가공제 | 일급과 승인된 무급휴가 일수 |

## 월할 계산

입사일이 급여월 안에 있으면 해당 월의 재직일수 비율로 기본급과 수당을 감액합니다.

```text
월할 비율 = 재직일수 / 해당 월 총 일수
월할 금액 = 전체 월 금액 × 월할 비율
```

## 비과세 처리

| 항목 | 기준 |
| --- | --- |
| 식대 | 설정된 식대 비과세 한도까지 비과세 |
| 교통비 | 자가 차량 보유 직원에게만 비과세 한도 적용 |

## 검증해야 할 부분

- 보험 요율은 적용 연도별로 관리해야 합니다.
- 간이세액표는 실제 운영 전 최신 기준으로 교체해야 합니다.
- 회사별 수당 정책은 고정 코드보다 설정 테이블로 분리하는 편이 안전합니다.
- 이미 승인된 급여 실행은 재계산 정책을 별도로 둬야 합니다.
